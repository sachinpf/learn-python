# Learn Python — The Complete Reference

A friendly, exercise-heavy Python curriculum for absolute beginners — designed for **Grades 3 to 10**, but useful for anyone who wants to go from zero to building real AI applications.

Every technical term is introduced in English with its Marathi (मराठी) equivalent in parentheses on first use, so learners can build mental anchors in their mother tongue.

## How to use this curriculum

1. **Start with [Part 1: Foundations](01_foundations/README.md).** Read every lesson, type out the examples yourself, and do the exercises at the end.
2. **Then Part 2.** Pick whichever format suits your mood: drill exercises, themed packs, curiosity essays, or puzzle challenges. Mix and match.
3. **Part 3 onwards** assumes you've finished Parts 1 and 2. Each part stands on the shoulders of the one before.

> **For parents and teachers:** every lesson is a single markdown file. You can read it on GitHub, in any editor, or print it out. No special platform required.

---

## What makes this different

- **No prior coding experience needed.** We start at "what is a program?".
- **Bilingual key terms.** *variable (चल)*, *function (फंक्शन)*, *loop (लूप)*. The first time a word appears, you'll see its Marathi gloss.
- **Show first, name later.** Every concept comes with code you can run before we explain the theory.
- **Exercises with full solutions.** Try first, peek when you're stuck.
- **One curriculum, all the way to AI.** Many beginner courses stop at loops and lists. This one keeps going.

---

## Requirements

- **Python 3.x** — free download from [python.org](https://www.python.org/downloads/)
- **A code editor** — PyCharm (Community Edition is free) is recommended in Part 1; later parts work with any editor (VS Code, Thonny, even online editors)
- **Curiosity, patience, and willingness to make mistakes.** Errors are how you learn.
