# GLOSSARY — Marathi Terms with Definitions

A single lookup for every technical term used in this curriculum, with its Marathi translation and a short Marathi definition. This is the canonical source — when in doubt, check here first.

> **Note for the author (Sachin):** the Marathi definitions below were drafted by an English-first assistant and **must be reviewed and corrected by you**. Look for clumsy phrasing, wrong word choices, or definitions that won't make sense to a 10-year-old. The "First lesson" column tells you where each term is first introduced — handy when reviewing a specific lesson's terminology.

**Inline gloss format used in lessons:** on first use in a lesson, the term appears as `term (Marathi — definition)`. After first use, the parenthetical is dropped. For trivial connectors (`and`, `or`, `not`), the definition column doubles as a brief plain-language explanation.

## Style reminders (matching `STYLE.md`)

- **Prose only.** Marathi never appears inside code or comments.
- **First mention rule.** A term gets its full `term (Marathi — definition)` gloss on **first use within a lesson**. Subsequent uses in the same lesson are English-only.
- **Don't invent translations.** If a term you need isn't in this glossary, add an entry here first (with definition) before using it in any lesson.
- **Variable names stay English.** Even when prose uses Marathi, the actual Python identifier stays in English (`student_age`, not `विद्यार्थी_वय`).
