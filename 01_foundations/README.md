# Part 1: Foundations — Step by Step Guide

A beginner-friendly Python course for students in **Grades 3 to 10**.
Each lesson builds on the previous one, with detailed explanations, many examples, and exercises with solutions.

> This is **Part 1** of a 5-part curriculum. See the [top-level README](../README.md) for what comes next (practice exercises, games, AI basics, AI applications).

## A note for absolute beginners

Some lessons include **Marathi (मराठी) translations** of key technical terms in parentheses on first use, like `variable (चल)` or `function (फंक्शन)`. This is to help learners anchor each new word in their mother tongue. The code, comments, and exercises stay in English so they remain runnable for everyone.

You'll also find **"common worry"** and **"watch out"** callouts throughout — these are designed to clear up the gotchas that confuse most beginners (the difference between `=` and `==`, why `input()` always returns a string, why `range(5)` stops at 4, the difference between `print()` and `return`, and more).

## How to Use This Guide

1. Open **PyCharm** on your computer (any editor works, but PyCharm is recommended for Part 1)
2. Create a new project called `learn-python`
3. Start with **Lesson 01** and work your way up
4. For each lesson, create a new Python file (e.g., `lesson01.py`) and try every example
5. Do the exercises at the end of each lesson before moving to the next one
6. When you finish all 12 lessons, head to **Part 2** for more practice (or jump back to the [top-level README](../README.md) to pick your next adventure)

## Tips for Success

- **Type the code yourself** — don't just read it. You learn by doing!
- **Make mistakes** — errors are how you learn. Read the error messages carefully.
- **Experiment** — change the examples and see what happens.
- **Do every exercise** — try on your own first, then check the solution.
- **Have fun** — programming is like solving puzzles!
